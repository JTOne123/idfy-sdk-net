# Idfy .NET SDK

## Installation
Using NuGet is the easiest way to install the SDK.

Package Manager:

	PM > Install-Package Idfy.SDK

Command line:  

	nuget install Idfy.SDK

.NET Core CLI:  

	dotnet add package Idfy.SDK
